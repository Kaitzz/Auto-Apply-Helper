# Extension Icons

Please add the following icon files:

- icon16.png (16x16 pixels)
- icon48.png (48x48 pixels)  
- icon128.png (128x128 pixels)

These are required for the Chrome extension to display properly.

## Quick Option

You can generate simple placeholder icons using any image editor or online tool.
For a professional look, consider using:
- Figma
- Canva
- https://favicon.io/

## Suggested Design

A simple briefcase or document icon in blue (#2563eb) would match the UI theme.
